Get started with your first build in just a few steps.

<Steps>
  <Step>
    ### Describe Your Idea

    Use the XO prompt bar to describe what you want to build. Think of it like briefing your smartest engineer — the clearer your idea, the better your results.

    **Example prompts:**
    - "Build a crypto portfolio tracker with wallet integration"
    - "Create a web app for booking yoga classes with admin dashboard"
  </Step>

  <Step>
    ### Let XO Assemble Your Team

    XO dynamically assembles a team of AI agents — Designer, Frontend Engineer, DevOps, Crypto Coder — to work on your project. No setup required.
  </Step>

  <Step>
    ### Watch the Build Unfold

    Each agent performs specialised tasks:

    - UI design from screenshots
    - Frontend development in Next.js, React, or Vue
    - Smart contract deployment on Polygon or any EVM chain
    - Cloud infrastructure setup and CI/CD automation
  </Step>

  <Step>
    ### Test & Iterate

    XO gives you a live demo, deployable code, and cloud execution environments. Give feedback, iterate, or download the codebase anytime.
  </Step>
</Steps>
