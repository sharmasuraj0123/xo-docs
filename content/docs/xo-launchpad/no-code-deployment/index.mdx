XO Launchpad supports three deployment methods, each suited to a different workflow.

## Deployment Methods

**[Deploying from Local](/docs/xo-launchpad/no-code-deployment/deploying-from-local)** — Build and push a Docker image from your local machine. Full control, no GitHub required.

**[Deploying from GitHub Repository](/docs/xo-launchpad/no-code-deployment/deploying-from-github-repository)** — Connect your repo and deploy straight from a branch. Ideal for CI/CD workflows.

**[Deploying from Lovable](/docs/xo-launchpad/no-code-deployment/deploying-from-lovable)** — Built a prototype on Lovable? Sync it to GitHub and deploy on XO Launchpad in three steps.
