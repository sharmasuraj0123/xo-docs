XO Vibe supports a wide range of application types — from simple landing pages to complex decentralised applications.

## App Types

- Full-stack Web Apps (React/Next.js)
- Decentralised Applications (DApps)
- Smart Contract Platforms
- Internal Automation Tools
- Landing Pages
- Agentic AI Workflows

## Core Capabilities

| Capability | Description | Maturity |
|---|---|---|
| **Frontend / UI** | Build user interfaces & frontend | Mature |
| **Persistence** | Store and retrieve data | Mature |
| **Authentication** | Handle user login and accounts | Experimental |
| **Backend endpoint** | API key protected endpoint such as OpenAI | Mature |
| **Deployment** | Publish, custom domains & deploy | Mature |

Most capabilities are production-ready. Authentication is still in experimental phase — validate carefully before shipping.
