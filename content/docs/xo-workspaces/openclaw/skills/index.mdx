Skills are add-on modules that give your OpenClaw agent new capabilities. Install them into your `~/.openclaw/workspace/skills` directory and they load automatically on the next session.

## Available Skills

| Skill | What It Does |
|-------|-------------|
| [Backup & Restore](/docs/xo-workspaces/openclaw/skills/backup-restore) | Encrypt, version, and restore your agent state via GitHub |
