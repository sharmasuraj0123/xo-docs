OpenClaw is an AI agent gateway you deploy on XO. Connect it to Telegram, give it your Anthropic API key, and your agent is live — no infrastructure to manage.

## Setup Walkthrough

<VideoEmbed id="a5a6481638a04ec49472a9b94c0959ab" title="OpenClaw Setup on XO Builders" />

## What You'll Need

- An [XO account](https://beta.xo.builders)
- An Anthropic API key
- A Telegram bot token (from [@BotFather](https://t.me/botfather))

## Deploy OpenClaw

<Steps>
  <Step>
    ### Create a New Project
    Go to [beta.xo.builders](https://beta.xo.builders).

    ![XO Builders dashboard](/images/openclaw-setup-01-dashboard.jpeg)

    Click the **+** icon next to **My Projects** in the left sidebar.

    ![Click the + icon to create a new project](/images/openclaw-setup-02-new-project-button.jpeg)

    Select **OpenClaw** and click **Create Agent**.

    ![Select OpenClaw and click Create Agent](/images/openclaw-setup-03-select-template.jpeg)
  </Step>
  <Step>
    ### Configure Project Details
    Fill in the project creation form:

    | Field | Value |
    |---|---|
    | **Project Name** | A unique name — e.g. `xo-openclaw` (note this down, you'll need it later) |
    | **Anthropic API Key** | Your Anthropic API key |
    | **Telegram Bot Token** | Your token from @BotFather |
    | **WhatsApp** | Optional — scan the QR code shown in the form to connect your agent to WhatsApp |
    | **Volume Size** | Default 2 GB is fine for most setups |

    You can add and remove channels as needed.

    ![Project creation form — optionally add WhatsApp](/images/openclaw-setup-04-add-whatsapp.jpeg)

    ![Remove a channel from your project](/images/openclaw-setup-05-remove-channel.jpeg)

    Click **Create Project**.

    ![Click Create Project to launch](/images/openclaw-setup-06-create-project.jpeg)
  </Step>
  <Step>
    ### Wait for Startup
    Your workspace takes **8–10 minutes** to provision. The OpenClaw environment is being set up in the background.

    <Callout type="info">
      Don't close the tab. You'll see the project status update when it's ready.
    </Callout>
  </Step>
  <Step>
    ### Open the OpenClaw Gateway
    Open the [Configure panel](/docs/xo-workspaces/core-platform-setup/access-restart-machine) and select **OpenClaw Gateway**. This opens the OpenClaw Dashboard in a new tab.

    ![Click OpenClaw Gateway](/images/openclaw-click-gateway.jpeg)
  </Step>
  <Step>
    ### Connect to the Gateway
    In the **Gateway Token** field, enter the exact project name you used in Step 2 (e.g. `xo-openclaw`).

    ![Enter Gateway Token](/images/openclaw-gateway-token.jpeg)

    Click **Connect**.

    ![Click Connect](/images/openclaw-click-connect.jpeg)

    The status indicator turns **green (OK)** when connected.
  </Step>
  <Step>
    ### Test Your Agent
    Navigate to the **Chat** tab in the sidebar. Send a message — try `hello`.

    ![Test your agent in chat](/images/openclaw-test-chat.jpeg)

    Your agent responds. OpenClaw is live.
  </Step>
</Steps>

<Callout type="warn">
  The Gateway Token must exactly match your project name from Step 2. A mismatch will prevent the dashboard from connecting.
</Callout>
