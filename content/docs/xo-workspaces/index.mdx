# XO Workspace

**The home for your AI Agent Team.**

XO Workspace lets you quickly and securely launch AI agents in an isolated remote execution environment — no local setup, no infrastructure to manage. Pick a template, create a project, and your agent is live.

---

## How it works

<Cards>
  <Card icon={<FolderPlus size={18} className="text-green-400" />} title="1. Create a Project">
    Every agent runs inside a project — a secure, sandboxed environment in the cloud.
  </Card>
  <Card icon={<Bot size={18} className="text-green-400" />} title="2. Choose your Agent">
    Pick a pre-configured agent template or bring your own custom agent.
  </Card>
  <Card icon={<Rocket size={18} className="text-green-400" />} title="3. Launch">
    Your agent spins up instantly in a secure remote execution environment, ready to work.
  </Card>
</Cards>

## Get Started

<Steps>
  <Step>
    ### Create an Account

    Sign up at [beta.xo.builders](https://beta.xo.builders/sign-up) using your GitHub, Google, or GitLab account. The Basic plan is free and gets you running immediately.

    [Full sign-up guide →](/docs/xo-workspaces/core-platform-setup/create-account)
  </Step>
  <Step>
    ### Create a New Project

    From your workspace dashboard, click **New Project**. Give it a name and you're ready to choose your agent.
  </Step>
  <Step>
    ### Choose your Agent

    You'll be presented with three options:

    <Cards>
      <Card
        icon={<Bot size={18} className="text-green-400" />}
        title="OpenClaw"
        description="A pre-configured OpenClaw agent — ready to deploy with built-in skills and integrations."
        href="/docs/xo-workspaces/agents/openclaw"
      />
      <Card
        icon={<Code2 size={18} className="text-green-400" />}
        title="Claude Code"
        description="A pre-configured Claude Code agent for AI-assisted software development."
        href="/docs/xo-workspaces/agents/claude-code"
      />
      <Card
        icon={<Wrench size={18} className="text-green-400" />}
        title="Custom Agent"
        description="Bring your own custom agent and run it in a secure remote environment. Full DIY control."
        href="/docs/xo-workspaces/agents/custom-agent"
      />
    </Cards>

    <Callout type="info">
      More agent templates are coming soon. OpenClaw and Claude Code are available now.
    </Callout>
  </Step>
</Steps>

## Why XO Workspace?

<Cards>
  <Card icon={<ShieldCheck size={18} className="text-green-400" />} title="Secure by Default">

  </Card>
  <Card icon={<Rocket size={18} className="text-green-400" />} title="Zero Setup">
